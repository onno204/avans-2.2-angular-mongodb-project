De Avans Appstore!

Wat is De Avans Appstore?

De Avans Appstore is voor de studenten van Avans om hun zelf gemaakten applicaties op een centraal punt te kunnen delen met elkaar.
Bij iedere app kan je ook een rating en een reactie toevoegen. Hierdoor kan je snel zien welke app het nou echt goed doet!
Vind jij een app nou echt leuk? Voeg deze dan toe aan je favorieten!

Wie kan mij app allemaal zien?

Iedereen van zowel binnen als buiten Avans kan de apps zien. Wil je liever je app nog niet openbaar hebben? Dan kan je hem op priv√© zetten!

Wie kan mijn app-details beheren?

Degene die de app aangemaakt heeft is ook de beheerder. Voor de rest kan niemand jouw details aanpassen.

Hoe laat ik een reactie achter?

Onderaan de app-details pagina kan je een reactie achterlaten.

Hoe voeg ik mijn eigen OS to aan de keuzelijst?

Alleen een admin kan een OS/apparaat toevoegen aan het systeem.

Hoe word ik een admin?

Een andere admin kan mensen hun rollen beheren.
